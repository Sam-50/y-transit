# **App Name**: SafeRoute

## Core Features:

- Driver Dashboard: Driver's dashboard provides a clean interface for managing routes and marking student attendance and sharing their location when dropping or picking up students.
- Parent Dashboard: Parent's dashboard enables real-time tracking of the bus location and management of children profiles.
- Admin Dashboard: Admin dashboard centralizes route management, student data, and communication tools.
- Safety Reporting: A form for reporting safety incidents, accessible to all users, that ensures issues are logged and tracked.
- Route Safety Analysis: AI-powered tool that analyzes incident reports to flag potentially unsafe routes or driver behaviors.
- Real-time Alerts: Utilize push notifications to alert parents about bus arrival times, delays, and safety alerts.

## Style Guidelines:

- Primary color:yellow to evoke trust and security.
- Background color: Off-white (#F9FBFC) to provide a clean and neutral backdrop.
- Accent color: Sky blue (#85C1E9) to highlight important actions and information.
- Body and headline font: 'Inter', sans-serif, for a modern, machined, objective, neutral look; suitable for both headlines and body text
- Use clear, recognizable icons from a set like FontAwesome for navigation and key features.
- Employ a grid-based layout with consistent spacing for a structured and easily navigable interface.
- Subtle transitions and animations for loading states and interactions to enhance user experience.
